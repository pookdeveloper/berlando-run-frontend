# PRD — Luxury Trail Running Apparel Platform

## 1. Vision

Create a premium digital-first brand selling technical running apparel designed for trail and all-terrain environments. The platform combines high-performance textiles, minimalist luxury aesthetics, and a strong global running community.

Core idea:

**Running as freedom.**

The brand positions running not only as sport but as exploration, culture, and identity.

---

## 2. Brand DNA

### Positioning

| Dimension | Position |
|---|---|
| Performance | High |

| Price | Premium |
| Community | Strong |
| Innovation | Textile technology |

### Brand Pillars

1. **Freedom** – running as exploration.
2. **Function** – every garment has a technical purpose.
3. **Form** – minimal luxury design.

### Example Manifesto

Running is not routine.  
It is a way to escape, explore terrain, and find rhythm.  

We design equipment for runners seeking **freedom, not limits**.

## 4. Product Strategy

### Categories

- Trail Shorts
- Technical Tees
- Base Layers
- Outerwear
- Hydration Accessories
- Running Caps
- Socks

### Technical Features

Each product should include multiple performance attributes.

| Feature | Description |
|---|---|
| 4-way stretch | Full mobility |
| Ultralight | <150 g |
| Quick dry | Fast moisture evaporation |
| Anti-odor | Merino or silver fiber |
| Laser ventilation | Breathability |

### Product Philosophy

Design rules:

1. Remove unnecessary elements
2. Maximize freedom of movement
3. Use premium performance materials
